# 工作约定

> 用途：记录当前 harness 的基本协作原则。

---

## 当前工作约定

1. 最外层 `D:\Nagi's Heart` 是唯一 workspace。
2. 当前 Git 仓库根目录是唯一活跃项目。
3. `notebook/` 与 `notebook-app/` 是冻结项目，默认不读、不改。
4. Ant大小姐的所有输入，不论大小，一律先到 PM 一一。
5. 执行者不直接承接来自 Ant大小姐的执行请求。
6. 只有写进 `task_board.md` 的任务，才算正式可执行任务。
7. 只有写进 `decision_log.md` 的拍板，才算正式生效决策。
8. 交班优先于发挥；没有交班，不开启新班。
9. 每个循环只推进一个受控小步骤，并在结束时回写结果。
10. 协作记录写在 `00_harness/`，代码和项目权威内容保留在仓库对应业务目录。
