# Skills 总入口

> 用途：作为 `00_harness/04_execution/skills/` 的统一入口，告诉不同角色该先看哪份 Skill。

---

## 一、怎么用

当某个角色准备开始一个新会话、接一个新任务、或恢复一个被打断的任务时：

1. 先读 `00_harness/00_project/working_agreement.md`
2. 再读自己角色对应的 Harness
3. 再读本目录下自己角色对应的 Skill
4. 再进入任务板、决策记录和实际项目文件

## 四、不要跳过的规则

1. 不要只看 Skill 不看 Harness
2. 不要只看聊天消息不看任务板
3. 不要把 Ant大小姐的临时话语直接当最终任务单
4. 不要先开工再补读规则
