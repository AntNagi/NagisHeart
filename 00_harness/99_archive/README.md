# PM Workspace

This folder centralizes PM-side collaboration materials that were previously split outside the main project repository.

Contents typically include:

- `PM_AGENT_INBOX/`
- `PM_AGENT_OUTBOX/`
- PM sync/status boards
- PM audit notes
- PM intake rollups
- short development or coordination logs

Usage rules:

1. Do not scan this folder by default for implementation-only tasks.
2. Read it when the task involves PM coordination, cross-agent traceability, audit history, or inbox/outbox messaging context.
3. Do not move large generated image assets out of inbox/outbox unless there is a specific cleanup task.
4. When referencing files here in messages or handoffs, use absolute paths to avoid confusion with the previous split-root layout.
