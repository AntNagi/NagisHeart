# 系统页设计交付 v1.0

## 页面边界

- `章节目录` 是传统章节列表，负责按篇章查看阅读进度。
- `Route / 记忆轨迹` 是另一张页面，负责展示路线、选择点、隐藏路线和进度，不可用 Route 替代章节目录。
- `系统设置`、`保存进度`、`回忆画廊` 已在同一份设计预览中补齐。

## 视觉统一

- 首页与系统页共用同一张主背景图，叠加深靛雾化层。
- 交互文字全部中文、居中或按内容轴对齐。
- 不使用长条按钮、胶囊按钮、切角按钮或按钮背景框。
- 设置项与存档位使用文字行和细分隔线；Gallery 的图片是内容，不是按钮。
- 颜色沿用深靛、银灰、暖金；不使用粉色主题。
